# Weathery-App
Weather App 
Shows the weather depending on your Location, or you can easily type the name of City and find out the weather conditions.
# Technologies & Features

- No storyboards
- Auto Layout
- Vector Assets
- Dark Mode
- Protocol-Delegate
- Parsing JSON
- HTTP/URLSession
- Weather API
- Error Handling
- Unit Testing
- CoreLocation
